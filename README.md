# 🚀 Navneet Mallick - Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and experience as a Computer Engineering student specializing in Web Development, Machine Learning, and Data Science.

## ✨ Features

- **Modern Design**: Sleek dark theme with gradient accents and smooth animations
- **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
- **Interactive Animations**: 
  - Floating particles background
  - Glowing profile picture with rotating ring
  - Parallax scrolling effects
  - Animated skill bars
  - Smooth section reveals
- **Theme Toggle**: Switch between dark and light modes
- **Project Showcase**: Interactive project cards with hover effects
- **Contact Form**: Integrated with EmailJS for direct messaging
- **Performance Optimized**: Fast loading with minimal dependencies

## 🛠️ Technologies Used

- HTML5
- CSS3 (Advanced animations, gradients, and transitions)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts (Poppins)
- EmailJS (Contact form integration)
